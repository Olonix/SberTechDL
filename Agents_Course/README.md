# Agents Course

https://huggingface.co/learn/agents-course/

Здесь собраны материалы по итогам прохождения курса. Все задачи были реализованы с использованием бесплатной модели `gemini-3.1-flash-lite`

### Введение Unit_1

Создание первого прототипа агента dummy_agent.py

### Знакомство с фреймворками Unit_2

Второй юнит был посвящен изучению трех инструментов: `smolagents`, `LangGraph` и `LlamaIndex`. Учебные ноутбуки были полностью переработаны под работу с гугловским API для взаимодействия с gemini

### Агент + RAG Unit_3

Создание агент с инструментом поиска по локальному датасету на базе `BM25Retriever`

### Финальный проект Unit_4

Итоговое задание заключалось в создании автономного агента для прохождения GAIA - бенчмарка для проверки способностей ИИ-ассистентов. Основной целью было преодолеть установленный порог 6/20 по количеству верных ответов. Количество верных ответов получилось 7/20 -> порог был пройден
