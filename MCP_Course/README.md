# MCP Course

https://huggingface.co/learn/mcp-course/

Здесь собраны материалы по итогам прохождения курса. Из-за недоступности Claude Code задачи были адаптированы под модель `gemini-3.1-flash-lite` и расширение `Continue` для VS Code 

В папке `.continue` лежат конфиги локальных моделей и MCP-серверов, использованных (или созданных) во время прохождения курса 

### Введение Unit_1

Изучение теоретической базы протокола: архитектура взаимодействия Host, Client и Server через JSON-RPC

### Компоненты протокола Unit_2

Изучение трех основных примитивов MCP: `Resources`, `Prompts` и `Tools`. Понимание механизмов передачи контекста от сервера к LLM

### Финальный проект Unit_3

(рабочая папка - `Unit_3/mcp_course/projects/unit3/`)

Создание сервера для автоматизации Pull Requests. Реализован анализ `git diff`-ов, прием вебхуков GitHub через `localtunnel` и отправка уведомлений в Slack
